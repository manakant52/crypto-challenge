Шифр Цезаря

Шифр Цезаря — это вид шифра подстановки, в котором каждый символ в открытом тексте заменяется символом, находящимся на некотором постоянном числе позиций левее или правее него в алфавите.

def caesar_cipher(text, shift): result = "" for char in text: if char.isalpha(): shift_base = 65 if char.isupper() else 97 result += chr((ord(char) - shift_base + shift) % 26 + shift_base) else: result += char return result # Пример использования encrypted = caesar_cipher("PYTHON", 3) print(encrypted) # Что будет выведено?
2

Base64

Base64 — это стандарт кодирования двоичных данных при помощи только 64 символов ASCII. Алфавит кодирования содержит текстово-цифровые символы A-Z, a-z и 0-9.

import base64 message = "Hello, World!" message_bytes = message.encode('ascii') base64_bytes = base64.b64encode(message_bytes) base64_message = base64_bytes.decode('ascii') print(base64_message) # Что будет выведено?
3

Шифр Атбаш

Шифр Атбаш — это простой шифр подстановки, который был создан для иврита. Он работает по принципу замены первой буквы алфавита на последнюю, второй на предпоследнюю и т.д.

def atbash_cipher(text): result = "" for char in text: if char.isalpha(): if char.isupper(): result += chr(90 - (ord(char) - 65)) else: result += chr(122 - (ord(char) - 97)) else: result += char return result # Пример использования encrypted = atbash_cipher("CRYPTO") print(encrypted) # Что будет выведено?
